# Rust Learning Projects

This repository contains a collection of Rust projects for personal practice and learning.

## Projects

### Guessing Game

A number guessing game that demonstrates:

- User input handling with `std::io`
- Random number generation using the `rand` crate
- Pattern matching and comparison operations
- Loop control and error handling
- Basic Rust syntax and control flow

**To run:**

```bash
cd guessing_game
cargo run
```

## Getting Started

Each project is a standalone Rust application with its own `Cargo.toml` file. Navigate to the project directory and use `cargo run` to build and execute the program.

## Requirements

- Rust and Cargo (latest stable version recommended)
- Install from [rustup.rs](https://rustup.rs/)
